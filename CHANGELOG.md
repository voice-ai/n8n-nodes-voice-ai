# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-01-28

### Added
- Initial release
- Text-to-speech conversion with customizable parameters (model, temperature, top_p, language)
- Voice management operations (list, get, create, update, delete)
- Support for MP3 and WAV audio formats
- Voice cloning from audio files
- Resource locator for easy voice selection
- Searchable voice list
